A machine learning classifier web app which can help us detect machines potentially infected by the DGA (Domain Generation Algorithm) malware. Typically, infected machines tend to generate a bunch of random domain names which will contain one active C&C server.
This application guides users on detecting potential DGA generated malware domains to avoid malware attacks, including for educational purposes.
